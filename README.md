
# MediMate – Intelligent Healthcare Management System

## Overview

MediMate is a comprehensive healthcare management system developed in C++ that integrates physical health tracking, mental health assessment, medication management, hospital discovery, and personalized health recommendations into a single application.

The system leverages Object-Oriented Programming principles, data-driven analytics, and location-based services to help users monitor and improve their overall well-being.

---

## Features

### Mental Health Assessment

* Implements the PHQ-9 mental health screening framework.
* Evaluates depression severity levels.
* Detects critical self-harm risk indicators.
* Generates severity-based assessments.

### Medication Reminder System

* Create and manage daily medication reminders.
* Create and manage weekly medication schedules.
* Prevents duplicate reminder entries.
* Stores reminder data using persistent file storage.

### Nutrition & Health Tracking

* Tracks daily calorie intake.
* Monitors protein, fat, and carbohydrate consumption.
* Supports food database search with intelligent matching.
* Allows custom food entries when database records are unavailable.

### Hospital Locator

* Finds nearby hospitals based on user pincode.
* Uses geographical coordinates for distance calculations.
* Ranks hospitals by proximity.
* Provides hospital contact and location information.

### Smart Health Advisor

* Analyzes:

  * Mental health score
  * Sleep duration
  * Activity level
  * Calorie intake
  * Mood levels
* Detects potential burnout patterns.
* Generates personalized health recommendations.
